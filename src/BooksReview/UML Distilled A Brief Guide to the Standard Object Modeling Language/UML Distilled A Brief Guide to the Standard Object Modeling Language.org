* Chapter 1. Introduction
** What Is the UML?
The UnifiedModelingLanguage (UML) is afamily of graphicalnotations, backed by single metamodel, that help indescribing and designing software systems, particularly software systemsbuilt 
using the object-oriented (OO) style. 

TheUML is arelatively open standard, controlled by the ObjectManagement Group(OMG), anopen 
consortium of companies. TheOMGwasformed tobuild standardsthat supported interoperability, 
specificallythe interoperability of object-oriented systems. TheOMG is perhapsbest knownforthe 
CORBA (CommonObjectRequest Broker Architecture) standards.
