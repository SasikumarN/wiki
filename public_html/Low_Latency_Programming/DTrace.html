Install DTrace in ubuntu

apt-get

sudo apt-get install systemtap-sdt-dev

This package provides dtrace.

git

git clone "https://github.com/dtrace4linux/linux.git" dtrace
cd dtrace
tools/get-deps.pl
make all 
make install
sudo make load
